Title: Add GitHub issue templates for structured work

## Problem
Issue creation is inconsistent, which makes delegation and agent execution less reliable.

## Goal
Add short issue templates that standardize problem framing, scope, and acceptance criteria.

## Scope
- Add cleanup task template
- Add portfolio signal task template
- Add Copilot agent task template
- Keep each template brief and implementation-ready

## Acceptance Criteria
- Templates exist under .github/ISSUE_TEMPLATE
- Each template includes problem, goal, scope, constraints, acceptance criteria, and agent notes
- Templates are short enough to fill out quickly
- Template language supports clear delegation

## Notes for Copilot coding agent
Favor structure over ceremony. The templates should reduce ambiguity, not add process overhead.

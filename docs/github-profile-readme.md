## Featured Projects

### [AI Todo List](https://github.com/dansbands/ai-todo-list)
Full-stack MERN app where each task calls an OpenAI Chat Completions endpoint to return a structured action plan, curated resources, and a suggested search query. Built with JWT auth, React, Express, and MongoDB. Demonstrates how AI can be embedded naturally into a workflow tool without being gimmicky.

### [danodeamedia.com](https://github.com/dansbands/dansbands.github.io)
My portfolio site — a continuously evolving Next.js/TypeScript application rebuilt from a legacy Wix site. Typed React components, App Router, EmailJS contact form, and a live résumé with collapsible sections. The codebase reflects how I approach frontend architecture: clean structure, no unnecessary dependencies.

Minimal Next.js App Router project that demonstrates an agentic workflow: user submits a goal → AI API returns structured tasks → tasks are rendered step-by-step with rerun/edit capability. Built to show product thinking, not just code.
